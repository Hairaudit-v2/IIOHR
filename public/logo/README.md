# IIOHR logo system

## Files

| File | Use |
|------|-----|
| `iiohr-mark.svg` | Crest only (emblem). Favicon, social, small contexts. |
| `iiohr-logo.svg` | Full logo: crest + “IIOHR” + “International Institute of Hair Restoration”. Light backgrounds. |
| `iiohr-monogram.svg` | “II” + follicle-O + “HR”. Avatars, app icons, tight spaces. |

## Light vs dark backgrounds

- **Light background**: use `iiohr-logo.svg`, `iiohr-mark.svg`, `iiohr-monogram.svg` as-is (stroke `#4A3526`, accent `#C8A47E`).
- **Dark background**: use the `-dark.svg` variants so the crest and type remain readable (light stroke/type).

## Specs

- **Crest**: Hair follicle cross-section (outer ring, shaft ring, dermal papilla node).
- **Stroke**: `#4A3526`, width `2`.
- **Accent (nodes)**: `#C8A47E`.
- **Typography**: Elegant serif (Cormorant Garamond / Georgia), slightly expanded tracking.

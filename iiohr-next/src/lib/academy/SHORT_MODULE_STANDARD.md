# Academy Short-Module Standard (Locked)

This standard is locked from two approved pilots:
- Consultant pilot: `role-and-scope-in-modern-hair-restoration-clinics`
- Nurse-suited pilot: `post-operative-support-follow-up-and-red-flag-escalation`

## Locked Sequence

1. Hero
2. Outcomes
3. Core teaching in 3-4 beats
4. One central visual teaching device
5. One principle/takeaway card
6. Recap and linked knowledge check
7. Safety and escalation
8. Compliance
9. Completion requirements

## Locked Components

- `ConsultantPilotLessonLayout`
- `LessonChapterNav` with benchmark pilot nav builder
- `RichTextAcademicBody` with `pilotBand` and `beatNumbers`
- `PilotScopeWorkflowVisual`
- `PilotPrincipleSummaryCard`
- `PilotPreAssessmentRecap`
- `LinkedAssessmentsCallout` inside recap/check closure container

## Flexible Elements (By Lesson Type)

- Beat titles and emphasis (foundation, pre-op, post-op, ethics, etc.)
- Central visual content (as long as there is one dominant visual device)
- Clinical application mix (communication examples, case card, practical task)
- Module context depth and examples density

## Authoring Rules: 3-4 Teaching Beats

- Author the core reading body with `##` headings for each beat.
- Target exactly 3-4 beats for short modules.
- Keep each beat focused and clinically scoped (no marketing framing).
- Keep pace tight: each beat should advance one clear teaching move.

## Target Study Time

- Short modules target `estimatedStudyMinutes: 30`.
